# Socket programming

Socket programming examples

- `tcp_echo_server/` ... echo server with TCP
- `udp_echo_server/` ... echo server with UDP
- `web_server/` ... web server

## Environments and tools

- GCC 11.4.0
- GNU Make 4.3
- clang-format 14.0.0
- Python 3.11.4
- Go 1.24.2
